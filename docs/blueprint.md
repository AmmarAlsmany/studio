# **App Name**: TeenMind

## Core Features:

- Onboarding & Login: Easy login using email/username and password. Age input to ensure appropriate content.
- Daily Symptom Tracker: Track mood, sleep, appetite, activity, and social engagement using engaging visual elements and optional journal entries.
- Progress Dashboard: Weekly/monthly graphs for mood trends, sleep, appetite, activity, and medication adherence.
- Smart Alerts & Support: Timed medication reminders with emojis and check-in prompts. 'Red Flag' alerts suggest contacting a doctor.
- Settings & Privacy: Option to export reports, passcode/Face ID, and optional parental access toggle with teen consent.
- AI Chat Support: AI tool that facilitates supportive conversations.
- Educational Resources: Provide section of curated videos and content about mental health and wellness topics

## Style Guidelines:

- Primary color: Calming blue (#64B5F6) to promote a sense of peace and stability.
- Background color: Light, desaturated blue (#E3F2FD) for a gentle and unobtrusive backdrop.
- Accent color: Soft lavender (#B39DDB) to provide a contrasting yet harmonious highlight for interactive elements.
- Teen-friendly fonts
- Minimalistic layout with smooth navigation
- Subtle animations for an engaging user experience
